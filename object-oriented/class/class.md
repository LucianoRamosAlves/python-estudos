# Classes em Python

As classes são um dos pilares da programação orientada a objetos (POO) e permitem criar objetos que possuem atributos (características) e métodos (comportamentos). Em Python, as classes são definidas usando a palavra-chave `class`.

## O que são métodos e atributos

- **Atributos**: São as características ou propriedades de um objeto. Por exemplo, em uma classe `Carro`, os atributos podem ser `cor`, `modelo` e `ano`.

- **Métodos**: São as funções que definem o comportamento de um objeto. Por exemplo, em uma classe `Carro`, os métodos podem ser `acelerar()`, `frear()` e `ligar()`.

## O que é uma instância de classe

Uma instância de classe é um objeto criado a partir de uma classe. Quando você cria uma instância, ela herda os atributos e métodos definidos na classe. Por exemplo, se você tiver uma classe `Carro`, pode criar uma instância chamada `meu_carro` que terá os atributos e métodos da classe `Carro`.

analogia para facilitar o entendimento: pense em uma classe como um molde ou uma receita, e a instância como o objeto real criado a partir desse molde ou receita. Por exemplo, a classe `Carro` é como um molde para criar carros, e cada carro específico que você cria (como `meu_carro`) é uma instância desse molde.
